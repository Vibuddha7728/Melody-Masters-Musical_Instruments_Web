## 🚀 Core Features

* 🤖 **AI Melody Assistant:** A custom-built, animated floating chatbot offering real-time guidance and FAQ support.
* ✨ **Dynamic UI:** Interactive product catalogs integrated with **AOS (Animate On Scroll)** for a premium feel.
* 🔐 **RBAC (Role-Based Access Control):** Specialized workflows and dashboards for **Admins, Staff,** and **Customers**.
* 📦 **Inventory Management:** Full CRUD system for managing instruments, categories, and real-time stock levels.
* 🛒 **E-commerce Engine:** Smooth user journey including a dynamic shopping cart and streamlined checkout.
* 📱 **Responsive Design:** Fully optimized for mobile, tablet, and desktop using **Bootstrap 5**.

---

## 🛠️ Technical Architecture

### **Frontend Stack**
* **HTML5 & CSS3:** Custom keyframe animations and theme control.
* **JavaScript (ES6+):** Chatbot logic and asynchronous UI updates.
* **Bootstrap 5:** Grid-based responsive layout.
* **AOS Library:** Scroll-triggered animations.

### **Backend & Database**
* **PHP:** Server-side logic, session handling, and secure authentication.
* **MySQL:** Relational database for persistent storage.

---

## 🏗️ Software Engineering Principles
* **MVC Influence:** Modular structure separating logic from presentation.
* **Database Normalization:** Efficient Relational Database Design for data integrity.
* **CRUD Operations:** Standardized methods for data handling.
